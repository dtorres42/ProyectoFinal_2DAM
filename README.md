PROYECTO INTERMODELUDAR

Este proyecto es un sistema basado en Visión Artificial y una arquitectura cliente-servidor para monitorizar espacios en tiempo real. Permite el control de aforo y la detección de objetos específicos mediante cámaras IP, mostrando la información en una aplicación móvil

## Tecnologías utilizadas
* **IA y Servidor:** Python, YOLOv8, OpenCV
* **Base de Datos:** Firebase
* **App Móvil:** Flutter
